README: SYSTEM HOSTINGU SERWERÓW MINECRAFT

WSTĘP

Witaj w oficjalnej dokumentacji systemu hostingowego do serwerów Minecraft. Ten dokument zawiera wszystkie niezbędne informacje oraz ostrzeżenia związane z konfiguracją i bezpieczeństwem systemu. Prosimy o zapoznanie się z poniższymi informacjami, aby zapewnić bezbłędne działanie Twojego serwera.

UWAGI OGÓLNE

WAŻNE OSTRZEŻENIE:

Zmiana jakiejkolwiek treści w plikach systemowych lub prób manipulacji plikami systemowymi prowadzi do natychmiastowego wyłączenia serwera oraz utraty danych!

  • Nie modyfikuj tego pliku!
  • Zmiana jakiejkolwiek struktury folderów skutkuje błędem systemu.
  • Nie używaj tego pliku do celów innych niż jego zapisywanie lub archiwizowanie.
  • System automatycznie loguje wszelkie manipulacje plikiem.

FUNKCJE SYSTEMU

System hostingowy oferuje następujące funkcje zarządzania serwerami Minecraft:

  1. Automatyczne aktualizacje serwera.
  2. Przełączanie między wersjami Minecrafta. OD (1.21) DO (1.21.x)
  3. Monitorowanie stanu serwera w czasie rzeczywistym. TYMCZASOWO TYLKO DLA WŁAŚCICIELA!!
  4. Wsparcie dla serwerów TYLKO Minecraft JAVA
  5. Wbudowane mechanizmy zabezpieczające przed atakami DDoS.

WYMAGANIA SYSTEMOWE

Wymagania dotyczące systemu operacyjnego i oprogramowania:

  • System operacyjny: Ubuntu 20.04 LTS lub wyższy, Debian 11, CentOS 7+
  • Minimalna pamięć RAM: 4 GB
  • Minimalna przestrzeń dyskowa: 20 GB SSD
  • Porty wymagane do działania: 25565 (Minecraft)

PROCEDURY ZABEZPIECZENIA

Dokumentacja pokazuje, jak skonfigurować serwer z odpowiednimi zabezpieczeniami:

  1. Używaj tylko bezpiecznych pluginów: Wybieraj tylko pluginy z zaufanych źródeł.
  2. Aktualizuj system regularnie: Używaj mechanizmów automatycznych do aktualizacji systemu operacyjnego.
  3. Twórz regularne kopie zapasowe

NOTATKI KOŃCOWE

Upewnij się, że Twój serwer działa zgodnie z najlepszymi praktykami bezpieczeństwa. Monitoruj logi systemowe, aby wykrywać wszelkie anomalie.

W przypadku problemów skontaktuj się z naszym zespołem wsparcia technicznego, używając panelu kontaktowego na naszej stronie.
